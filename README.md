## Как использовать конфиг

1. **Скопируйте** любую ячейку `Конфиг` из таблицы выше (длинная строка `vless://…` / `vmess://…` / `ss://…`).
2. **Вставьте** её в совместимый с V2Ray клиент:
   - **Android:** v2rayNG, NekoBox, Hiddify Next, Husi.
   - **iOS:** Streisand, Shadowrocket, Hiddify, FoXray.
   - **Windows:** v2rayN, NekoRay, Hiddify, Clash Verge.
   - **macOS:** V2Box, NekoRay, ClashX Meta, Hiddify.
   - **Linux:** NekoRay, Hiddify, Clash Meta.
3. **Подключайтесь.** Если узел медленный или не работает, попробуйте следующий — свежие конфиги появляются каждые несколько минут.

## Частые вопросы (FAQ)

### Эти VPN / прокси-конфиги действительно бесплатны?

Да. Они собираются из публично опубликованных Telegram-каналов и публикуются здесь бесплатно, без регистрации и рекламы. Содержимое этого README передано в общественное достояние по [лицензии The Unlicense](https://unlicense.org/).

### Как часто обновляется список?

Этот `README.md` переписывается **каждые 10 минут** и содержит 30 самых свежих конфигов. Для обновлений в реальном времени подпишитесь на [@abc_configs](https://t.me/abc_configs) в Telegram, где новые конфиги появляются через несколько секунд после сбора.

### Какие протоколы поддерживаются?

V2Ray (VMess), VLESS (включая Reality), Trojan, Shadowsocks (SS) и MTProto. Любой стандартный URI, начинающийся с `vmess://`, `vless://`, `trojan://`, `ss://`, или ссылка Telegram MTProto поддерживаются основными клиентами.

### Конфиг не работает, почему?
