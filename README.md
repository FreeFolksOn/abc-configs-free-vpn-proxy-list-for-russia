## Что такое Abc Configs?

**Abc Configs** — это поддерживаемый сообществом, автоматически обновляемый список **бесплатных VPN и прокси-серверов**, собранный из множества публичных Telegram-каналов. Конфиги — это короткие URI-строки (`vless://`, `vmess://`, `trojan://`, `ss://` и т. д.), которые можно вставить в любой совместимый с V2Ray клиент и сразу подключиться к бесплатному прокси-серверу.

Этот репозиторий GitHub содержит один файл — этот `README.md`, который наш бот переписывает **каждые 10 минут**, добавляя 30 самых свежих конфигов. Для обновлений в реальном времени (новые конфиги каждые несколько секунд) подпишитесь на нас в Telegram: [@abc_configs](https://t.me/abc_configs).

## Частые вопросы (FAQ)

### Эти VPN / прокси-конфиги действительно бесплатны?

Да. Они собираются из публично опубликованных Telegram-каналов и публикуются здесь бесплатно, без регистрации и рекламы. Содержимое этого README передано в общественное достояние по [лицензии The Unlicense](https://unlicense.org/).

### Как часто обновляется список?

Этот `README.md` переписывается **каждые 10 минут** и содержит 30 самых свежих конфигов. Для обновлений в реальном времени подпишитесь на [@abc_configs](https://t.me/abc_configs) в Telegram, где новые конфиги появляются через несколько секунд после сбора.

### Какие протоколы поддерживаются?

V2Ray (VMess), VLESS (включая Reality), Trojan, Shadowsocks (SS) и MTProto. Любой стандартный URI, начинающийся с `vmess://`, `vless://`, `trojan://`, `ss://`, или ссылка Telegram MTProto поддерживаются основными клиентами.

### Конфиг не работает, почему?

У бесплатных прокси короткий и непредсказуемый срок жизни. Узел, работающий сейчас, через минуту может быть перегружен или заблокирован. Просто попробуйте другую строку из таблицы или возьмите более свежий конфиг в [@abc_configs](https://t.me/abc_configs). Список постоянно обновляется.

### Безопасно ли использовать эти прокси?

Это сторонние серверы, управляемые неизвестными лицами, поэтому относитесь к ним как к любому бесплатному VPN: **не** входите в важные аккаунты, **не** передавайте незашифрованный трафик и предпочитайте конфиги с TLS (VLESS+TLS / Trojan / Reality). Используйте их для обхода цензуры, а не для передачи конфиденциальных данных.
